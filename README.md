# view-count-badge
Attach this badge on your website to statistics page views.  
## Requirement
python3  
Redis
##Usage
`pip install -r ./requirements.txt`    
`python start.py`    
Goto [http://127.0.0.1:12000/?key=your_unique_key](http://127.0.0.1:12000/?key=your_unique_key)
